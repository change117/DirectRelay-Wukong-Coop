  1. Navigate to HOST/ folder
  2. Double-click DirectRelay.exe
  3. GUI control panel appears

  1. Navigate to CLIENT/ folder
  2. Double-click DirectRelayConnect.exe
  3. Enter host's IP address when prompted
  4. Click connect
  5. Game launches automatically
